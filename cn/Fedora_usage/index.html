☆ Fedora 中文乱码

升级Fedora 28 发现终端乱码，经过一翻排查后发现是语言支持升级的时候不知道怎么 被卸载了。

执行下面的命令可以解决这个问题：
sudo dnf install langpacks-zh_CN
