如何在 macOS 中连接 VNC 服务器

发表于2025-08-29|更新于2025-08-29

MacOS 自带 VNC 客户端，不用使用第三方软件。

打开 Finder , 快捷键打开 command + k ，在地址栏输入 vnc://1.1.1.1:590x

若 vnc 以 vncserver :2 启动，则需要在地址栏输入 vnc://1.1.1.1:5902
